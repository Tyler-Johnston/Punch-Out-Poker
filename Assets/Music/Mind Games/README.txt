== Pack Structure ==
Each song contains 3 rendered versions:
* Full loop (ready to drag & drop audio file)
* Folder with stems divided by instrument.
* Folder with sequences divided into 16 bar sections (except for Brainstorm & Clear Headed which are 32 bar sections because they're 180 BPM).
